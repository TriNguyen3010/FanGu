# Fan Gu 🎮

Demo game viral cho Gen Z — *fan nhau vì cùng gu*.

Trả lời câu hỏi dí dỏm → ghép với người **cùng gu** (kiểu dating nhẹ) → share thách bạn bè → leo bảng xếp hạng săn **vé concert** sao hạng A. Gameplay nhẹ, dễ tích hợp vào app sẵn có; viral đến từ **nội dung** chứ không từ cơ chế chơi.

## Nội dung repo

| File | Mô tả |
|------|-------|
| [`FanGu_Demo.html`](FanGu_Demo.html) | Demo mobile (iPhone dọc) **bấm chơi được** — mở bằng trình duyệt. Luồng: Câu hỏi → Kết quả (vote cộng đồng) → Cùng gu (gửi tim / say hi / kết bạn) → Bảng xếp hạng (vé concert) → Chia sẻ & thách đấu KOL. |
| [`FanGu_TrinhSep.html`](FanGu_TrinhSep.html) | One-pager **trình sếp**: 5 trụ cột chiến lược, vòng lặp viral, chỉ số đo & điều kiện dừng, xử lý phản biện, đề xuất thử nghiệm. |

## Cách mở

Mở trực tiếp file `.html` bằng trình duyệt (Chrome/Safari). Cần kết nối mạng để tải font & thư viện.

## Lưu ý

- Tên **Fan Gu** = *Fan* (fan idol / fan nhau) + *Gu* (gu cá nhân).
- Nội dung số liệu, người chơi, phần thưởng trong demo là **placeholder minh hoạ**.
- Vé concert & hợp tác KOL là **mục tiêu đàm phán**, chưa phải cam kết đã chốt.
