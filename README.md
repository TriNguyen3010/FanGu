# Fan Gu 🎮

Demo game viral cho Gen Z — *fan nhau vì cùng gu*.

Trả lời câu hỏi dí dỏm → ghép với người **cùng gu** (kiểu dating nhẹ) → share thách bạn bè → leo bảng xếp hạng săn **vé concert** sao hạng A. Gameplay nhẹ, dễ tích hợp vào app sẵn có; viral đến từ **nội dung** chứ không từ cơ chế chơi.

## Demo

[`index.html`](index.html) — demo mobile (iPhone dọc) **bấm chơi được**.

Luồng: Câu hỏi → Kết quả (vote cộng đồng) → Cùng gu (gửi tim / say hi / kết bạn) → Bảng xếp hạng (vé concert) → Chia sẻ & thách đấu KOL.

Mở trực tiếp `index.html` bằng trình duyệt (Chrome/Safari) — cần mạng để tải font & thư viện.

## Lưu ý

- Tên **Fan Gu** = *Fan* (fan idol / fan nhau) + *Gu* (gu cá nhân).
- Số liệu, người chơi, phần thưởng trong demo là **placeholder minh hoạ**.
- Vé concert & hợp tác KOL là **mục tiêu đàm phán**, chưa phải cam kết đã chốt.
